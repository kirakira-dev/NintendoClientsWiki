[LDN](LDN-Protocol) allows games to specify a passphrase. This is required to establish a connection with the network. This page lists the passphrase for some games:

| Game | Passphrase |
| --- | --- |
| Mario Kart 8 Deluxe | `MarioKart8Delux` followed by 17 null bytes |
| Super Mario Maker 2 | `LunchPack2DefaultPhrase` |
| Tomodachi Life: Living the Dream | `ABCDEFGHIJKLMNOPQRSTUVWXYZ` |
